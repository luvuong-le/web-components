Shadow DOM

Encapsulation

Allows for encapsulation. You can think of shadow DOM as a scoped subtree inside your element. Allows you to hide the implementation of the custom element from other users The difference is that none of the code inside a shadow DOM can affect anything outside it, allowing for handy encapsulation.
